Motor Design Lab
An interactive 3D explorer and design lab for ECM/BLDC motors.

Setup:

Place index.html, style.css, and script.js in the same folder.

Open index.html in any web browser.

Requires an internet connection to load Three.js and Chart.js.

Features:

3D Explorer: Interactive motor visualization.

Design Lab: Live performance calculations and charts.

Formulas: Physics equation reference.
